## Local Context

`ep0043` concerns ordinary additive Sidon sets `A, B subset {1, ..., N}` satisfying
`(A-A) cap (B-B) = {0}`. The target quantities are

- `binom(|A|, 2) + binom(|B|, 2)`
- `f(N)`, the maximum size of an additive Sidon set in `{1, ..., N}`

The problem is in the additive/difference-set Sidon setting.

## Source Check

The candidate paper is real and relevant to a different Sidon-adjacent topic:

- arXiv: https://arxiv.org/abs/1801.08733
- PDF mirror: https://real.mtak.hu/99395/1/An_improved__IJNT.pdf

The arXiv abstract defines a multiplicative 3-Sidon set using the equation
`s1 s2 s3 = t1 t2 t3` and proves an upper bound for multiplicative 3-Sidon subsets of
`{1, 2, ..., n}`.

The PDF introduction explicitly distinguishes additive Sidon sets from multiplicative
Sidon sets and then studies the multiplicative `k`-Sidon problem.

## Assessment

Do not approve `ep0043-r001`.

Reasons:

- The paper is about multiplicative product equations, not additive sums or difference
  sets.
- `ep0043` needs information about two additive Sidon sets with disjoint difference
  sets.
- The candidate is a keyword hit on `Sidon` and `upper bound`, but it does not appear
  to give a direct method for the EP43 formulation.

This is useful negative data for the literature filter.

## Experiment Lesson

The review system needs context-aware Sidon filtering. For additive-combinatorics Sidon
problems, papers about multiplicative Sidon sets, completely Sidon sets in operator
spaces, and algebraic-geometry Sidon sets should be flagged or scored lower unless the
problem statement asks for that setting.
